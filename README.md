# SQL-customer-transaction-analysis
SQL project analyzing customer transaction data to identify revenue trends, and regional performance using MYSQL

##Dataset Description 

The dataset contain 40 customer transaction recors with the following fields:
- Trancation_ID
- Customer_ID
- Transaction_Date
- Region
- Amount

- The data is used to peform business analysis through Sql  queries.
- -------
## Tool Used
-MY SQL
-Github

## Business Question Solved 

The Project answer the following  Business questions :

1. Calculate total revenue.
2. Find revenue by region
3. count total transactions.
4. Calculate average transction  value.
5. Find the highest transactions.
6. Find the lowest transcations.
7. Count transactions by region
8.Find customer with transcation  above the average  amount.
9. Find monthly revenue.
10. Find revenue generated each month.
11. Find the average revenue  by region.
12. Identify the regions with revenue  above the average.
13. Calculate each region's contribution to total revenue.
14. Find the maximum transaction in each region.
15. Find the minimum transaction in each region.

-----------
##key insights 
-Total business revenue was calculated using sql aggregate functions.
-Regional revenue comparison identified the  highest  - peforming region.
- Monthly revenue trends were  analyzed for business reporting.

- average  transaction values helped  measure customer spending
- regional contribution analysis highlighted  the percentage share of total revenue.
